# Meridian
### by Reach Point Research

<img src="./public/rpr-logo-dark.jpg" alt="Reach Point Research" height="40" />

A modern, institutional-grade financial research terminal. Free, open source, and built for serious investors.

## Features
- Real-time (delayed) quotes and price charts
- Full financial statements — income, balance sheet, cash flow
- Earnings calendar with EPS estimates and actuals
- Options pricing calculator with Black-Scholes and full Greeks
- Macro market overview sidebar
- Company news feed

## Data Sources
- [Financial Modeling Prep](https://financialmodelingprep.com) — fundamentals, financials, news, earnings
- [Polygon.io](https://polygon.io) — price history and market data

## Getting Started

1. Clone the repo and install dependencies
```bash
npm install
```

2. Add your API keys
```bash
cp .env.example .env
# Edit .env and add your FMP and Polygon API keys
```

3. Start the development server
```bash
npm start
```
